CLAUDE.md 이해하기

Claude의 "기억 노트"예요. 매번 같은 설명을 반복하지 않아도 되게 해줍니다.

📓 비유로 이해하기

새 인턴이 출근하면, 첫날에 "우리 팀은 이런 일을 하고, 보고서는 이 형식으로 쓰고, 이건 이렇게 해줘"라고 알려주잖아요? CLAUDE.md는 바로 그 인수인계 문서예요. 한 번 만들어두면, Claude가 매번 대화를 시작할 때 자동으로 읽어서 여러분의 업무 방식을 기억합니다.

📍 CLAUDE.md의 종류

global
~/.claude/CLAUDE.md
모든 프로젝트에 적용되는 개인 설정. '항상 한국어로 답변해줘' 같은 것
project
프로젝트폴더/CLAUDE.md
특정 프로젝트에만 적용. '이 프로젝트는 마케팅 분석용이야' 같은 것
local
~/.claude/projects/.../memory/
Claude가 작업하면서 자동으로 배운 것들. 직접 관리할 필요 없어요

📋 CLAUDE.md 템플릿

CLAUDE.md
# 프로젝트 소개 - 이 프로젝트는 [프로젝트 설명]입니다 - 주요 목적: [목적] # 작업 규칙 - 모든 결과물은 한국어로 작성 - 보고서 형식은 마크다운(.md) 사용 - 파일명은 날짜_제목.md 형식으로 # 폴더 구조 - data/ : 원본 데이터 파일 - reports/ : 생성된 보고서 - templates/ : 템플릿 파일 # 자주 하는 작업 - 매주 월요일: 주간 보고서 생성 - 매월 1일: 월간 데이터 분석 # 주의사항 - .env 파일은 절대 수정하지 말 것 - 원본 데이터 파일은 삭제하지 말 것
