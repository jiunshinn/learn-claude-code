🚀 시작하기

처음 시작하기

설치했다면, 이제 첫 대화를 시작해 볼까요?

1

작업할 폴더로 이동하기

Claude Code는 현재 위치한 폴더의 파일들에 접근해요. 먼저 작업하고 싶은 폴더로 이동하세요.

Mac

cd ~/Documents/내프로젝트

Windows

cd C:\Users\내이름\Documents\내프로젝트
💡 Mac에서는 폴더를 터미널로 드래그하면 경로가 자동 입력돼요. Windows에서는 폴더의 주소 표시줄에 powershell을 입력하면 해당 위치에서 바로 열 수 있어요.
✨ VS Code 사용자라면 더 쉬워요! 작업할 폴더를 VS Code로 열고(File → Open Folder), Ctrl + `으로 내장 터미널을 열면 자동으로 해당 폴더에서 시작돼요. cd 명령어를 입력할 필요가 없어요!
2

Claude Code 실행하기

claude 라고 입력하고 Enter를 누르세요. 첫 실행 시 계정 연결 화면이 나와요.

claude
3

프로젝트 초기화하기

/init을 입력하면 Claude가 폴더를 분석하고 CLAUDE.md라는 '기억 노트'를 만들어요. 다음 대화부터 더 똑똑해집니다!

/init
4

첫 번째 대화하기

이제 한국어로 자유롭게 대화하세요! Claude가 파일을 읽거나 수정할 때 허락을 구하니, 확인 후 승인하면 돼요.

이 폴더에 어떤 파일들이 있는지 정리해줘
!

권한 승인 이해하기

Claude Code는 안전을 위해 파일을 읽거나 수정하기 전에 반드시 여러분의 허락을 구해요. Y를 누르면 승인, N을 누르면 거부할 수 있어요. 어떤 작업을 하려는지 설명을 읽고 판단하세요.
