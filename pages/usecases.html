실전 활용 사례

이런 것도 되나요? 네, 됩니다. 프롬프트를 복사해서 바로 써보세요.

🧾

경비 보고서 자동 생성

신용카드 내역 CSV를 넣으면, 카테고리별로 정리된 경비 보고서를 만들어줘요.

프롬프트 예시
expenses.csv 파일을 읽어서 다음을 해줘: 1. 날짜순으로 정렬 2. 카테고리별로 분류 (식비, 교통, 숙박 등) 3. 카테고리별 소계와 총합 계산 4. expense_report.md 파일로 보고서 작성
🔍

경쟁사 분석 리포트

경쟁사 목록을 주면, 각 회사의 특징을 조사해서 비교표를 만들어줘요.

프롬프트 예시
@competitors.md 파일에 있는 경쟁사 5개를 다음 기준으로 비교 분석해줘: - 핵심 기능 - 가격 정책 - 타겟 고객 - 장단점 결과를 comparison_table.md로 저장해줘
📂

파일 정리 및 관리

어질러진 폴더를 종류별, 날짜별로 깔끔하게 정리해줘요.

프롬프트 예시
Downloads 폴더를 정리해줘: - 이미지 → images/ - 문서(PDF, DOC) → documents/ - 스프레드시트 → spreadsheets/ - 나머지 → others/ 각 폴더 안에서 날짜별로 하위 폴더도 만들어줘
📈

콘텐츠 성과 분석

SNS 게시물 데이터를 분석해서 인사이트를 뽑아줘요.

프롬프트 예시
@instagram_data.csv를 분석해서: 1. 가장 반응이 좋았던 게시물 TOP 10 2. 요일별·시간대별 참여율 패턴 3. 해시태그별 평균 도달 수 4. 다음 달 콘텐츠 전략 제안 결과를 content_report.md로 저장해줘
✉️

마케팅 뉴스레터 작성

업데이트 내용을 주면 뉴스레터 초안을 만들어줘요.

프롬프트 예시
@updates.md 파일의 이번 달 업데이트 내용으로 뉴스레터를 작성해줘: - 톤: 친근하고 전문적 - 구조: 인사 → 주요 업데이트 3개 → CTA - 길이: 500자 내외 - newsletter_march.md로 저장
📝

회의록 정리 및 액션 아이템 추출

회의록 텍스트를 넣으면 깔끔하게 정리하고 할 일을 추출해요.

프롬프트 예시
@meeting_notes.txt를 정리해줘: 1. 참석자 목록 2. 주요 논의 사항 (3줄 요약) 3. 결정 사항 4. 액션 아이템 (담당자 + 기한) meeting_summary.md로 저장해줘
