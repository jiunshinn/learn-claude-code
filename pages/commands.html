핵심 명령어 가이드

용도별로 정리했어요. 모두 외울 필요 없이, 필요할 때 찾아 쓰세요.

/clear 필수

대화 내용을 초기화해요. 새로운 작업을 시작할 때 꼭 사용하세요.

💡 다른 주제로 넘어갈 때 /clear를 하면 Claude가 헷갈리지 않아요
/compact 필수

대화가 길어졌을 때, 핵심만 남기고 압축해요. 대화가 느려질 때 사용하세요.

💡 "대화가 길어져서 느려졌다" 싶으면 /compact
/help 필수

사용 가능한 모든 명령어를 보여줘요. 뭘 해야 할지 모를 때!

claude -c 유용

마지막 대화를 이어서 계속할 수 있어요. 터미널을 껐다 켜도 OK!

💡 실수로 터미널을 닫았을 때 claude -c로 복구
/init 필수

프로젝트를 분석하고 CLAUDE.md를 만들어요. 새 프로젝트에서 처음 한 번만 실행하세요.

/memory 유용

CLAUDE.md 파일을 직접 편집할 수 있어요. Claude에게 기억시킬 내용을 수정할 때 사용해요.

/model 유용

사용할 Claude 모델을 바꿀 수 있어요. 더 빠르거나 더 똑똑한 모델을 선택하세요.

/doctor 유용

설치 문제를 진단해요. Claude Code가 이상하게 동작할 때 사용하세요.

단축키 기능 언제 쓰나요?
Ctrl + C 현재 작업 취소 Claude가 너무 오래 걸릴 때
Ctrl + D Claude Code 종료 작업을 다 끝냈을 때
Esc 현재 동작 중지 잘못된 방향으로 갈 때
Esc × 2 되돌리기 지점 표시 이전 상태로 돌아가고 싶을 때
Tab 확장 사고 모드 복잡한 작업 전에 Claude가 더 깊이 생각하게 하고 싶을 때
Shift + Tab 자동 승인 모드 전환 매번 승인하기 귀찮을 때 (주의해서 사용!)
@파일명 필수

특정 파일을 대화에 포함시켜요. Claude가 그 파일을 직접 참고합니다.

예: "이 @report.csv 파일을 분석해줘"
@폴더명/ 유용

폴더 전체를 참조해요. 여러 파일을 한꺼번에 보여줄 때 편해요.

예: "@data/ 폴더의 모든 CSV를 합쳐줘"
파이프 | 유용

다른 명령어의 결과를 Claude에게 바로 전달할 수 있어요.

예: cat memo.txt | claude -p "요약해줘"
