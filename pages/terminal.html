🖥️ 터미널

CLI / 터미널이란?

Claude Code를 사용하려면 터미널이 필요해요. 어렵지 않아요 — 차근차근 알아봐요.

⌨️

CLI (Command Line Interface)

텍스트 명령어로 컴퓨터와 대화하는 방식이에요. 마우스 클릭(GUI) 대신 키보드로 모든 걸 제어할 수 있어요.

  • ls — 파일 목록 보기
  • cd — 폴더 이동하기
  • git — 코드 버전 관리하기
  • GUI보다 빠르고 자동화하기 쉬움
🖥️

터미널

CLI를 실행하는 프로그램이에요. 컴퓨터와 대화하는 "창문" 같은 것이라고 생각하면 돼요.

  • Mac: Terminal.app, iTerm2
  • Windows: PowerShell, Windows Terminal
  • 터미널 안에서 shell(zsh, bash)이 명령어를 해석
  • Claude Code도 이 터미널 안에서 실행돼요

🎯 비유로 이해하기

터미널은 "컴퓨터와 대화하는 채팅창"이에요. 보통은 마우스로 폴더를 클릭해서 열지만, 터미널에서는 cd Documents라고 입력하면 같은 일이 일어나요. Claude Code는 바로 이 채팅창 안에서 AI와 대화하며 일하는 도구입니다.

💡 터미널이 어렵다면? VS Code를 활용하세요!

별도의 터미널 앱을 열 필요 없이, VS Code 안에서 바로 터미널을 사용할 수 있어요.

📝

VS Code란?

Microsoft에서 만든 무료 코드 편집기예요. 개발자뿐만 아니라 비개발자도 파일을 편집하고 관리하는 데 많이 사용해요.

  • 완전 무료, 가볍고 빠름
  • 파일/폴더를 시각적으로 볼 수 있음
  • 터미널이 내장되어 있음
  • Claude Code 확장도 설치 가능
🎯

왜 VS Code 터미널이 좋을까?

검은 화면만 보이는 별도 터미널 앱과 달리, VS Code에서는 파일 목록을 보면서 터미널을 사용할 수 있어요.

  • 왼쪽에 파일/폴더가 보여서 안심
  • 파일 클릭하면 바로 내용 확인 가능
  • 터미널 + 편집기가 한 화면에
  • 폴더를 드래그해서 열면 자동으로 경로 설정

🖥️ VS Code에서 터미널 여는 방법

VS Code를 설치하고 폴더를 열었다면, 아래 방법으로 내장 터미널을 열 수 있어요.

🍎 Mac

단축키: Ctrl + ` (백틱)

또는 상단 메뉴 → Terminal → New Terminal

🪟 Windows

단축키: Ctrl + ` (백틱)

또는 상단 메뉴 → Terminal → New Terminal

VS Code — 내장 터미널
탐색기
📁 내프로젝트
📄 report.md
📄 data.csv
📄 CLAUDE.md
TERMINAL
$ claude
✨ Claude Code에 오신 걸 환영합니다!

↑ VS Code에서는 왼쪽에 파일 목록이 보이면서 아래에서 터미널을 사용할 수 있어요

🚀 VS Code로 시작하는 3단계

1. code.visualstudio.com에서 VS Code를 다운로드하고 설치해요.

2. 작업할 폴더를 VS Code로 열어요. (폴더를 VS Code 아이콘에 드래그하거나, File → Open Folder)

3. Ctrl + ` 을 눌러 터미널을 열고, claude를 입력하면 끝!

왜 Claude Code인가요?

터미널에서 AI와 직접 협업하면 이런 점이 좋아요.

1

IDE 전환 없이 바로 작업

터미널에서 바로 AI와 협업해요. 다른 프로그램을 왔다 갔다 할 필요가 없어요.

2

파일을 직접 다뤄요

파일 읽기, 편집, 검색, 실행을 AI가 직접 수행해요. 파일을 업로드하거나 복붙할 필요 없어요.

3

Git과 자연스럽게 통합

코드 변경 사항을 추적하고, 커밋하고, 브랜치를 관리하는 Git 워크플로우와 매끄럽게 연결돼요.

4

외부 도구 연결 (MCP)

MCP 서버를 통해 브라우저, DB, API 등 외부 도구를 Claude에 연결할 수 있어요.

5

대화형으로 복잡한 작업 처리

컨텍스트를 유지하며 여러 단계의 복잡한 작업을 대화하듯 처리할 수 있어요.
