꿀팁 모음

Claude Code를 더 잘 쓰는 방법들이에요.

01

🎯 프롬프트 구조화하기

막연하게 말하지 말고, 구조를 갖춰서 요청하세요. 결과물이 훨씬 좋아져요.

작업: [무엇을 할지] 맥락: [배경 정보] 형식: [원하는 결과물 형태] 조건: [지켜야 할 것] 저장: [어디에 저장할지]
02

🔄 주제가 바뀌면 /clear

다른 작업을 시작할 때는 반드시 /clear로 대화를 리셋하세요. Claude가 이전 맥락에 혼란을 겪지 않아요.

03

📎 @로 파일 직접 지정

'이 파일'이라고 말하는 대신 @파일명으로 정확히 지정하세요. Claude가 정확한 파일을 참조합니다.

@data/sales.csv 분석해줘
04

🧠 Tab으로 깊이 생각하게

복잡한 분석이 필요할 때, 메시지 입력 전 Tab을 누르면 Claude가 '확장 사고 모드'로 더 깊이 고민해요.

05

🔍 탐색 → 계획 → 실행

큰 작업은 세 단계로 나누세요. 먼저 현황 파악, 그 다음 계획 수립, 마지막으로 실행. 한 번에 다 하려 하지 마세요.

06

🔒 안전하게 사용하기

비밀번호나 API 키는 CLAUDE.md에 적지 마세요. 파일 수정 전 Claude의 계획을 꼭 확인하고, 중요한 파일은 백업해 두세요.
