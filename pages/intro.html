💡 소개

Claude Code가 뭔가요?

쉽게 말하면, 내 컴퓨터에 사는 AI 비서예요. 웹 브라우저의 Claude(claude.ai)와 달리, 내 컴퓨터의 파일을 직접 읽고 쓸 수 있죠.

💬

웹 Claude (claude.ai)

브라우저에서 대화하는 AI. 파일을 직접 업로드해야 하고, 한 번에 처리할 수 있는 양에 제한이 있어요.

  • 파일을 직접 업로드해야 함
  • 대화 길이에 제한이 있음
  • 간단한 질문이나 짧은 작업에 적합
  • 별도 설치 없이 바로 사용 가능
⚡

Claude Code (터미널)

내 컴퓨터에서 동작하는 AI. 폴더와 파일에 직접 접근하고, 복잡한 작업을 오래 처리할 수 있어요.

  • 내 파일에 직접 접근 (업로드 불필요)
  • 파일 크기·개수 제한 없음
  • 복잡한 작업을 몇 시간이고 처리 가능
  • 반복 작업 자동화에 최적

🎯 비유로 이해하기

웹 Claude가 "카페에서 만나는 컨설턴트"라면, Claude Code는 "내 사무실에 출근하는 인턴"이에요. 인턴은 내 서류함을 직접 열어보고, 파일을 정리하고, 보고서를 작성해서 저장까지 해줘요. 여러분은 "이거 해줘"라고 말만 하면 됩니다.
