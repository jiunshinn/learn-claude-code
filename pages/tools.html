🧰 유용한 도구

유용한 도구 소개

Claude Code와 함께 쓰면 생산성이 올라가는 도구들을 소개합니다.

🖥️ 터미널 도구

Claude Code는 터미널에서 동작해요. 이 도구들을 함께 쓰면 터미널 작업이 훨씬 편해집니다.

01

tmux — 터미널 분할기

하나의 터미널 창을 여러 개로 나눠서 쓸 수 있어요. Claude Code를 실행하면서 다른 창에서 파일을 확인하거나, 서버 로그를 동시에 볼 수 있습니다.

brew install tmux
Ctrl+B 후 "로 화면 분할, %로 세로 분할. 세션이 유지되어 SSH 연결이 끊겨도 작업이 사라지지 않아요.
02

gh CLI — GitHub 명령어 도구

GitHub의 거의 모든 기능을 터미널에서 사용할 수 있어요. PR 생성, 이슈 관리, 저장소 클론까지 브라우저 없이 가능합니다.

brew install gh && gh auth login
gh repo clone 사용자/저장소, gh pr create, gh issue list 등 자주 쓰는 명령어만 알면 충분해요.
03

iTerm2 / Windows Terminal

기본 터미널보다 훨씬 편한 터미널 앱이에요. 탭 분할, 검색, 자동완성, 테마 변경 등 편의 기능이 풍부합니다.

brew install --cask iterm2
Mac은 iTerm2, Windows는 Microsoft Store에서 Windows Terminal을 설치하세요.

✏️ 에디터 & 개발 환경

Claude Code가 만든 파일을 보거나 수정할 때 쓰면 좋은 도구들이에요.

04

VS Code — 코드 에디터

Microsoft의 무료 코드 에디터. Claude Code와 함께 파일을 보고 편집할 때 유용해요. 확장 프로그램이 풍부하고, 터미널도 내장되어 있어요.

brew install --cask visual-studio-code
터미널에서 code .을 입력하면 현재 폴더를 VS Code로 열 수 있어요.
05

Cursor — AI 코드 에디터

AI 기능이 내장된 코드 에디터. VS Code를 기반으로 만들어져서 익숙하고, Claude Code와 병행해서 쓰면 시너지가 커요.

Claude Code로 큰 작업을 하고, Cursor로 세부 수정을 하는 워크플로우가 효율적이에요.
06

Homebrew / Chocolatey

프로그램 설치를 한 줄 명령어로 해결하는 패키지 매니저. Node.js, Python 등 필수 도구 설치에 유용해요.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Mac은 Homebrew, Windows는 Chocolatey를 사용하세요. brew install 패키지명으로 간편 설치.

🚀 데이터 & 생산성

Claude Code와 함께 쓰면 작업 효율이 크게 올라가는 도구들이에요.

07

jq — JSON 데이터 뷰어

JSON 데이터를 터미널에서 보기 좋게 정리해 줘요. API 응답을 분석하거나 설정 파일을 확인할 때 유용합니다.

brew install jq
cat data.json | jq '.'으로 JSON을 예쁘게 출력. jq '.name'으로 특정 필드만 추출 가능.
08

MCP 서버 — 기능 확장 플러그인

Claude Code의 기능을 확장하는 플러그인 시스템이에요. 웹 검색, 데이터베이스 연결, 파일 시스템 접근 등 다양한 기능을 추가할 수 있습니다.

Claude Code 설정에서 MCP 서버를 연결하면, Claude가 외부 도구와 직접 소통할 수 있어요.
09

GitHub Desktop — Git GUI

Git을 GUI로 쉽게 관리할 수 있어요. 터미널 명령어 없이 커밋, 브랜치, PR을 시각적으로 관리할 수 있습니다.

brew install --cask github
gh CLI가 터미널 중심이라면, GitHub Desktop은 시각적으로 변경사항을 확인하고 싶을 때 좋아요.
10

Wispr Flow — 음성 딕테이션

말로 하면 텍스트로 바꿔주는 음성 입력 도구예요. Claude Code에 긴 프롬프트를 타이핑하는 대신 말로 설명하면 훨씬 빠르고 편합니다.

어디서든 음성으로 텍스트 입력이 가능해요. Claude Code 프롬프트 작성, 문서 작성 등 타이핑이 많은 작업에 특히 유용합니다.

💡 추천 조합

기본 세트: iTerm2 + tmux + Claude Code — 터미널 하나로 모든 걸 해결하는 조합이에요.
개발 세트: VS Code + gh CLI + Claude Code — 코드를 보면서 Claude Code로 작업하고, gh로 바로 PR까지.
풀 세트: Cursor + tmux + gh CLI + jq + Claude Code — 가장 강력한 조합. 모든 작업을 효율적으로 처리할 수 있어요.
