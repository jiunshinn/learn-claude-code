🔧 설치

설치하기

5분이면 충분해요. 차근차근 따라해 보세요.

📌 시작하기 전에 필요한 것

Claude Pro 구독 (월 $20)이 필요해요. claude.ai에서 가입할 수 있어요. API 키를 따로 구매할 수도 있지만, 구독이 가장 간편합니다.

💡 터미널이 무섭다면? 다른 방법도 있어요!

터미널 설치가 부담스러우시다면, 더 쉬운 방법도 있어요:

  • 데스크톱 앱 — 일반 프로그램처럼 설치해서 사용할 수 있어요. Windows 다운로드 | Mac 다운로드
  • 웹 버전 — 브라우저에서 바로 사용: claude.ai/code
  • VS Code 확장 — VS Code를 사용한다면 확장 마켓플레이스에서 "Claude Code"를 검색하세요.

✨ VS Code 내장 터미널로 설치하기 (비개발자 추천!)

별도 터미널 앱을 여는 게 어렵다면, VS Code의 내장 터미널을 사용해 보세요. 파일 목록을 보면서 터미널을 쓸 수 있어서 훨씬 편해요.

  1. VS Code를 설치하고, 작업할 폴더를 열어요 (File → Open Folder)
  2. 키보드에서 Ctrl + ` (백틱)을 눌러 내장 터미널을 열어요
  3. 아래 설치 명령어를 그대로 복사해서 붙여넣으면 돼요!

💡 VS Code 터미널에 대해 더 알고 싶다면 → 터미널 페이지를 확인해 보세요.

1

터미널 열기

Command(⌘) + Space를 눌러 Spotlight를 열고, "터미널" 또는 "Terminal"을 입력한 후 Enter를 누르세요.

2

설치 명령어 입력

아래 명령어를 복사해서 터미널에 붙여넣고 Enter를 누르세요:

curl -fsSL https://claude.ai/install.sh | bash

Homebrew를 사용하고 있다면 이 방법도 가능해요:

brew install claude-code
3

설치 확인

설치가 완료되면 아래 명령어로 확인하세요:

claude --version
4

Claude Code 시작

이제 실행해 보세요!

claude
⚠️ 사전 준비: Windows에서는 Git for Windows가 필요할 수 있어요. 미리 설치해 주세요.
1

PowerShell 열기

키보드에서 Win 키를 누르고 "PowerShell"을 입력한 후 Enter를 누르세요. 또는 Win + X를 눌러 메뉴에서 "Windows PowerShell"을 선택하세요.

💡 붙여넣기 팁: PowerShell에서는 마우스 오른쪽 클릭으로 붙여넣기가 됩니다.

2

설치 명령어 입력

아래 명령어를 복사해서 PowerShell에 붙여넣고 Enter를 누르세요:

irm https://claude.ai/install.ps1 | iex

CMD(명령 프롬프트)를 사용한다면:

curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd

WinGet을 사용할 수도 있어요:

winget install Anthropic.ClaudeCode
3

설치 확인

설치가 완료되면 아래 명령어로 확인하세요:

claude --version
4

Claude Code 시작

이제 실행해 보세요!

claude
1

터미널 열기

Ctrl + Alt + T를 눌러 터미널을 열어주세요.

2

설치 명령어 입력

아래 명령어를 복사해서 터미널에 붙여넣고 Enter를 누르세요:

curl -fsSL https://claude.ai/install.sh | bash
3

설치 확인 및 시작

설치가 완료되면 확인하고 실행하세요:

claude --version
claude

🖥️ 터미널이 처음이라면? — 완전 초보 가이드

🍎 Mac에서 터미널 열기

  1. Command(⌘) + Space를 눌러요
  2. "터미널" 또는 "Terminal"을 입력해요
  3. Enter를 누르면 터미널이 열려요

🪟 Windows에서 터미널 열기

  1. 키보드에서 Win 키를 눌러요
  2. "PowerShell"을 입력해요
  3. Enter를 누르면 터미널이 열려요

📝 알아두면 좋은 기본 명령어

cd Documents

→ Documents 폴더로 이동해요

ls

→ 현재 폴더의 파일 목록을 보여줘요 (Windows에서는 dir)

cd ..

→ 상위 폴더로 이동해요
